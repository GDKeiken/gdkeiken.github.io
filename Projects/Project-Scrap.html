Project Scrap

Goal

To create a fully functional RPG framework built on the Unreal 4 Engine. Providing all the basic implementation for a working rpg (character, classes, equipment, etc), as well as a providing features like a Dialogue, and save system that can be used with other project.

The end result should be a working prototype of a turn based rpg.

Features

Save System

An extension of the existing save system in Unreal 4, currently providing the following features:

  • Generic object serialization
  • Full blueprint support and implementation
  • Events for saving and loading
  • Async saving

RPG Library

  • Characters
  • Combat system
  • Quest system
  • Items and equipment
